Métodos de Instalação


Instalação via CDN

Para incluir Bootstrap em seu projeto, precisamos adicionar dois arquivos:

  • Bootstrap
  • Bootstrap JavaScript

A parte JavaScript do Bootstrap são dependentes do JQuery. Então precisamos do arquivo da biblioteca JQuerry JavaScript também.
Todos esses arquivos que serão incluidos no seu projeto podem ser diretamente baixados de um CDN (Content Delivery Network, ou Rede de Entrega de Conteúdo traduzido literalmente).

    https://code.jquery.com/
    http://getbootstrap.com/getting-started/
    https://www.bootstrapcdn.com
Abra o arquivo src/index.html e insira:
  • O elemento Link no topo do header para incluir o arquivo CSS do Bootstrap
  • O elemento script para incluir o JQuery no fundo do corpo de texto (o elemento body)
  • O elemento script para incluir o arquivo do JavaScript do Bootstrap no fundo do corpo de texto (o elemento body)
Agora o seu index.html deverá se paracer com o seguinte código: 

<!doctype html>
<html>
<head>
  <meta charset="utf-8">
  <title>Angular Bootstrap Demo</title>
  <base href="/"><meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="icon" type="image/x-icon" href="favicon.ico">
  <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.1/css/bootstrap.min.css" integrity="sha384-WskhaSGFgHYWDcbwN70/dfYBj47jz9qbsMId/iRN3ewGhXQFZCSftd1LZCfmhktB" crossorigin="anonymous">
</head>
<body>
  <app-root>Loading...</app-root>
  <script src="https://code.jquery.com/jquery-3.1.1.min.js" integrity="sha256-hVVnYaiADRTO2PzUGmuLJr8BLUSjGIZsDYGmIJLv2b8=" crossorigin="anonymous"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>
</body>
</html>
Instalação via NPM
Outra maneira de adicionar o Bootstrap em seu projeto Angular é instalando na pasta do seu projeto usando NPM $ npm install bootstrap --save $ npm install jquery --save

Isso instala o Bootstrap e o jQuery na pasta node_modules dentro do diretório do projeto e garante que ambas as dependências também sejam adicionadas ao arquivo package.json.

Após ambos os pacotes terem sido instalados com sucesso, os arquivos jQuery e Bootstrap podem ser encontrados em:

  • node_modules/jquery/dist/jquery.min.js
  • node_modules/bootstrap/dist/css/bootstrap.min.css
  • node_modules/bootstrap/dist/js/bootstrap.min.js

    • Para adicionar esses arquivos ao projeto, você pode tanto:

    adicionar os elementos link e script correspondentes ao index.html, como fizemos antes, ou adicionando os caminhos dos arquivos no array de "styles" e "scripts" no arquivo .angular-cli.json:

    "styles": [
        "styles.css",
        "../node_modules/bootstrap/dist/css/bootstrap.min.css"
      ],
      "scripts": [
        "../node_modules/jquery/dist/jquery.min.js",
        "../node_modules/bootstrap/dist/js/bootstrap.min.js"
      ],
    Instalação do NG-Bootstrap

    O Ng-Bootstrap é baseado no Bootstrap 4 e pode ser adicionado ao seu projeto Angular da seguinte maneira.

    O Ng-Bootstrap está disponível como um pacote NPM, portanto, a instalação pode ser feita usando o seguinte comando no diretório do projeto:

    npm install --save @ng-bootstrap/ng-bootstrap

    Além disso, o Ng-Bootstrap exigia que o Bootstrap 4 fosse adicionado ao nosso projeto. Instale via:

    $ npm install bootstrap@4.0.0-alpha.6

    Agora adicione bootstrap.min.css, jquery.min.js e bootstrap.min.js ao arquivo .angular-cli.json, como fizemos antes.

    Uma vez instalado, você precisa importar o módulo principal do Ng-Bootstrap, NgbModule, do pacote @ng-bootstrap/ng-bootstrap. Adicione a seguinte instrução de importação para app.module.ts:

    import {NgbModule} from '@ng-bootstrap/ng-bootstrap';

    Em seguida, precisamos adicionar este módulo à matriz de importações do decorador @NgModule. Se você quiser importar o NgbModule no seu módulo raiz (por exemplo, AppModule), precisará chamar o método factory forRoot(), como você pode ver a seguir:

    @NgModule({
        declarations: [AppComponent, ...],
        imports: [NgbModule.forRoot(), ...],
        bootstrap: [AppComponent]
    })
    export class AppModule {
    }
